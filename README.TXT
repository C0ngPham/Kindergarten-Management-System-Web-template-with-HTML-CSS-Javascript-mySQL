-------------------How to run this demo-------------------
PLEASE FOLLOW THESE STEP!!!
============================
1. Download and store in folder "C:\\Program Files\Xampp\htdocs\KindergartenManagementSystem"
2. Open Xampp and press Start at Apache and MySQL.
3. Open your browser, go to http://localhost/phpmyadmin/
4. Create new database with name is KindergartenManagemtSystem
5. Import database 'schoolmanagement.sql'
6. Run the login page: http://localhost/KindergartenManagementSystem/index.php
7. Login by admin accout -> user: admin , password: admin
8. Enjoy our Demo!!!
