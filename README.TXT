-------------------How to run this demo-------------------
PLEASE FOLLOW THESE STEP!!!
============================
1. Open Xampp and press Start at Apache and MySQL.
2. Open your browser, go to http://localhost/phpmyadmin/
3. Create new database with name is KindergartenManagemtSystem
4. Import database 'schoolmanagement.sql'
5. Run the login page: http://localhost/KindergartenManagementSystem/index.php
6. Login by admin accout -> user: admin , password: admin
7. Enjoy our Demo!!!
